# AE2 Virus

AE2 Virus is a NeoForge mod concept for adding network-security and virus-infection mechanics to Applied Energistics 2 networks.

## Status

This repository is initialized as a clean Minecraft 1.21.1 NeoForge mod skeleton. The gameplay systems are still in design and early implementation planning.

## Development

```powershell
.\gradlew.bat build
```
